# site_dzaha

A simple Flask site for uploading files to Google Drive.

## Tech Stack

- **Python + Flask**

## Run locally

```bash
git clone https://github.com/maksimkaprosuperhacker/site_dzaha.git
cd site_dzaha
pip install -r requirements.txt
python main.py
```
